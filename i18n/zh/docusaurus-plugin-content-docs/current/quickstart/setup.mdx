### 步骤 1. 登录商家门户
访问【GGGPay商户门户】(https://merchant.gggpay.org) ,然后在 **侧边栏** 左起，进入 **开发管理** -> **开发设置**
![图片](/img/quickstart/get-configuration.jpeg)

### 步骤 2. 生成凭证
如果是商家第一次访问此页面，请填写*Redirect URL、Callback URL、Withdraw Callback URL*，并生成**Key、Secret**，以及**PublicKey**和**PrivateKey**的密钥对。

:::note
请妥善保管**私钥**，因为 GGGPay 不会存储私钥。
:::
